1. 프로그램 소개
학생 출결 관리 시스템은 학생의 출석, 지각, 결석 정보를 효율적으로 기록하고 조회할 수 있도록 제작한 데스크톱 기반 프로그램입니다.
학생은 로그인 후 자신의 출결 현황과 학적 정보를 조회할 수 있으며, 달력 UI를 통해 출결 상태를 한눈에 확인할 수 있습니다.
또한 로그인 시 자동으로 출석이 기록되는 기본 기능을 포함하여 사용 편의성을 높였습니다.

2. 사용 기술
- C#
- WPF
- SQLite
- MVVM 패턴
- GitHub

4. 주요 기능 설명
- 로그인/로그아웃 기능 (db사용)
- 로그인시 출결 기록 자동 저장 (db사용)
- 달력을 통한 출결 조회 (db사용)
- 학생 정보 관리 (db사용)
- 학생 비밀번호 변경 (db사용)

4. 실행 방법
4-1) Visual Studio2022에서 솔루션 실행 
4-2) 로그인 학번 : 20230395  비밀번호 : 1234
4-3) 사이드바에서 페이지 이동
4-4) 헤더에서 로그아웃 버튼 클릭시 로그아웃 

    
6. 개선하고 싶은 점
- 로그인 + 위치 기반(GPS) 출석 체크
- 월별 출석률을 분석해 출석률이 가장 높은 학생을 보여주는 기능을 추가하고 싶었습니다.
- 교직원 전용 페이지를 별도로 만들어 더 체계적인 출석 통계를 제공하고 싶습니다.
- 사용자 UI를 보다 직관적이고 깔끔하게 개선하고 싶습니다.
